# Project documentation service
## Features:
### Users:
- Создание пользователя
- Обновление пользователя
- Удаление пользователя
- Поиск по ID
- Поиск всех пользователей
### Pages:
- Создание страниц
- Обновление страниц

  При обновлении страницы, предыдущая версия сохраняется для версионирования
- Удаление старницы
- Поиск по ID
- Поиск всех страниц
### Blocks:
- Создание блоков
- Обновление блоков
- Удаление блоков
- Поиск по ID
- Поиск всех блоков
- Добавление к блокам картинок
### Tags:
- Создание тэгов
- Обновление тэгов
- Удаление тэгов
- Поиск по ID
- Поиск всех тэгов
### Comments:
- Создание комментов
- Обновление комментов
- Удаление коментов
- Поиск по ID
- Поиск всех комментов

В Main представлено классическое развитие событий:
1. Пользователь регистрируется
2. Пользователь создает страницу
3. Добавляет блоки, теги, пишет комменты
4. К блоку добавляет картину
